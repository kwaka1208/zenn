本書では、プログラミングを始めるために不可欠な3つのツール、**Git**、**Node.js**、**Python** のインストール方法を解説します。

Windows、macOS、Linux (Ubuntu想定) の3つの OS に対応しており、それぞれの環境に最適な方法を紹介します。

## 対象とする OS
- **Windows**: Windows 10/11 (PowerShell またはコマンドプロンプトを使用)
- **macOS**: Apple Silicon (M1/M2/M3) または Intel Mac
- **Linux**: Ubuntu などの Debian 系ディストリビューション

## 構築するツール
1. **Git**: バージョン管理システム
2. **Node.js**: JavaScript 実行環境（モダンな開発には必須）
3. **Python**: データ分析、AI、自動化に強い言語
   - Python については、プロジェクトごとに環境を分けるための「仮想環境」の構築方法についても触れます。

さあ、開発環境を整えていきましょう。
