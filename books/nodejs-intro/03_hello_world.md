---
title: "最初のプログラムを動かそう"
---

Node.jsをインストールできたら、さっそくJavaScriptを動かしてみましょう。

## 1. ファイルの作成
適当なフォルダを作成し、その中に `hello.js` というファイルを作成します。

```javascript:hello.js
console.log("Hello, Node.js!");
```

## 2. 実行
ターミナル（またはコマンドプロンプト）を開き、ファイルを作成したディレクトリで以下のコマンドを実行します。

```bash
node hello.js
```

## 3. 結果
画面に以下のように表示されましたか？

```text
Hello, Node.js!
```

おめでとうございます！これであなたはNode.jsエンジニアの第一歩を踏み出しました。
