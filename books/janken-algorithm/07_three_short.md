## 三人版のあいこはどう判定する？

二人版では「手が同じ = あいこ」でしたが、三人版のあいこは2種類あります。

1. **全員同じ手**（ぐー・ぐー・ぐー など）
2. **三すくみ**（ぐー・ちょき・ぱーが全種類出た）

この2つを一度に判定するにはどうすればよいでしょうか？ここで**リスト**を使うアイデアを紹介します。

## アイデア：手の種類をリストで管理する

「出た手の種類がいくつか」を調べるために、**フラグリスト**を作ります。

```python
hands_list = [0, 0, 0]
```

このリストの各インデックスは手に対応しています。

| インデックス | 対応する手 |
|---|---|
| `hands_list[0]` | ぐー（0）が出たかどうか |
| `hands_list[1]` | ちょき（1）が出たかどうか |
| `hands_list[2]` | ぱー（2）が出たかどうか |

誰かがその手を出したら、対応する場所を `1` にします。

```python
hands_list[player] = 1  # プレイヤーの手の場所を 1 にする
hands_list[cp1]    = 1  # コンピュータ1の手の場所を 1 にする
hands_list[cp2]    = 1  # コンピュータ2の手の場所を 1 にする
```

例えば「プレイヤー=ぐー、cp1=ちょき、cp2=ぐー」なら：

```python
hands_list = [0, 0, 0]
hands_list[0] = 1  # ぐーが出た
hands_list[1] = 1  # ちょきが出た
hands_list[0] = 1  # ぐーがまた出たが、すでに 1 なので変わらない
# 結果: hands_list = [1, 1, 0]
```

## sum() で出た手の種類数を調べる

`sum(hands_list)` で `hands_list` の合計を計算すると、**出た手の種類数**がわかります。

| `sum(hands_list)` | 意味 |
|---|---|
| `1` | 全員が同じ手を出した（あいこ） |
| `2` | 2種類の手が出た（勝ち負けが決まる） |
| `3` | 3種類全部出た = 三すくみ（あいこ） |

「あいこ」は `sum` が `1` または `3` のときです！

## プログラム

```python
import random

hands  = {0: "ぐー", 1: "ちょき", 2: "ぱー"}
player = random.randint(0, 2)
cp1    = random.randint(0, 2)
cp2    = random.randint(0, 2)

print(f"プレイヤーの手: {hands[player]}")
print(f"コンピュータ1の手: {hands[cp1]}")
print(f"コンピュータ2の手: {hands[cp2]}")
print("結果: ", end="")

# 出た手の種類をリストで管理する
hands_list = [0, 0, 0]
hands_list[player] = 1
hands_list[cp1]    = 1
hands_list[cp2]    = 1

result = sum(hands_list)  # 出た手の種類数を数える

if result == 1 or result == 3:
    # 全員同じ手 or 三すくみ → あいこ
    print("あいこ")
elif player == 0 and hands_list[1] == 1:
    # プレイヤーがぐー、かつちょきが出ている → 勝った
    print("勝った")
elif player == 1 and hands_list[2] == 1:
    # プレイヤーがちょき、かつぱーが出ている → 勝った
    print("勝った")
elif player == 2 and hands_list[0] == 1:
    # プレイヤーがぱー、かつぐーが出ている → 勝った
    print("勝った")
else:
    print("負けた")
```

## プログラムの解説

### あいこの判定

```python
if result == 1 or result == 3:
    print("あいこ")
```

`result == 1`（全員同じ）または `result == 3`（三すくみ）のとき「あいこ」です。27パターンのうち9パターンのあいこをこの1行で処理できます。

### 勝ちの判定

```python
elif player == 0 and hands_list[1] == 1:
```

「プレイヤーがぐー（0）で、ちょき（インデックス1）が出ている」なら、プレイヤーはぐーで相手にちょきがいる = 勝ちです。

| プレイヤーの手 | 勝ち条件 |
|---|---|
| ぐー（0） | `hands_list[1] == 1`（ちょきが出ている） |
| ちょき（1） | `hands_list[2] == 1`（ぱーが出ている） |
| ぱー（2） | `hands_list[0] == 1`（ぐーが出ている） |

### 負けの判定

```python
else:
    print("負けた")
```

あいこでも勝ちでもなければ「負け」です。`else` 1行でまとめられます。

## 27行 → 7行に！

| | 元のプログラム | 改善後 |
|---|---|---|
| `print` の数 | 27 | 5（あいこ・勝×3・負） |
| `if` の段数 | 3段ネスト | 1段のみ |

27パターンのif文が、たった数行のシンプルな構造に変わりました。
