## 27パターンをそのまま書く

二人版と同じように、組み合わせ表をそのまま `if` 文で書き起こします。

プレイヤーの手（3通り）× cp1の手（3通り）× cp2の手（3通り）の3段階ネストになります。

## プログラム

```python
import random

# 手の数値を文字列に変換するための辞書（表示用）
hands  = {0: "ぐー", 1: "ちょき", 2: "ぱー"}
player = random.randint(0, 2)  # プレイヤーの手
cp1    = random.randint(0, 2)  # コンピュータ1の手
cp2    = random.randint(0, 2)  # コンピュータ2の手

print(f"プレイヤーの手: {hands[player]}")
print(f"コンピュータ1の手: {hands[cp1]}")
print(f"コンピュータ2の手: {hands[cp2]}")
print("結果: ", end="")

# プレイヤーがぐー（0）
if player == 0:
    if cp1 == 0:
        if cp2 == 0:       #  1: ぐー vs ぐー   vs ぐー    → あいこ
            print("あいこ")
        elif cp2 == 1:     #  2: ぐー vs ぐー   vs ちょき  → 勝った
            print("勝った")
        elif cp2 == 2:     #  3: ぐー vs ぐー   vs ぱー    → 負けた
            print("負けた")
    elif cp1 == 1:
        if cp2 == 0:       #  4: ぐー vs ちょき vs ぐー    → 勝った
            print("勝った")
        elif cp2 == 1:     #  5: ぐー vs ちょき vs ちょき  → 勝った
            print("勝った")
        elif cp2 == 2:     #  6: ぐー vs ちょき vs ぱー    → あいこ（三すくみ）
            print("あいこ")
    elif cp1 == 2:
        if cp2 == 0:       #  7: ぐー vs ぱー   vs ぐー    → 負けた
            print("負けた")
        elif cp2 == 1:     #  8: ぐー vs ぱー   vs ちょき  → あいこ（三すくみ）
            print("あいこ")
        elif cp2 == 2:     #  9: ぐー vs ぱー   vs ぱー    → 負けた
            print("負けた")

# プレイヤーがちょき（1）
if player == 1:
    if cp1 == 0:
        if cp2 == 0:       # 10: ちょき vs ぐー   vs ぐー    → 負けた
            print("負けた")
        elif cp2 == 1:     # 11: ちょき vs ぐー   vs ちょき  → 負けた
            print("負けた")
        elif cp2 == 2:     # 12: ちょき vs ぐー   vs ぱー    → あいこ（三すくみ）
            print("あいこ")
    elif cp1 == 1:
        if cp2 == 0:       # 13: ちょき vs ちょき vs ぐー    → 負けた
            print("負けた")
        elif cp2 == 1:     # 14: ちょき vs ちょき vs ちょき  → あいこ
            print("あいこ")
        elif cp2 == 2:     # 15: ちょき vs ちょき vs ぱー    → 勝った
            print("勝った")
    elif cp1 == 2:
        if cp2 == 0:       # 16: ちょき vs ぱー   vs ぐー    → あいこ（三すくみ）
            print("あいこ")
        elif cp2 == 1:     # 17: ちょき vs ぱー   vs ちょき  → 勝った
            print("勝った")
        elif cp2 == 2:     # 18: ちょき vs ぱー   vs ぱー    → 勝った
            print("勝った")

# プレイヤーがぱー（2）
if player == 2:
    if cp1 == 0:
        if cp2 == 0:       # 19: ぱー vs ぐー   vs ぐー    → 勝った
            print("勝った")
        elif cp2 == 1:     # 20: ぱー vs ぐー   vs ちょき  → あいこ（三すくみ）
            print("あいこ")
        elif cp2 == 2:     # 21: ぱー vs ぐー   vs ぱー    → 勝った
            print("勝った")
    elif cp1 == 1:
        if cp2 == 0:       # 22: ぱー vs ちょき vs ぐー    → あいこ（三すくみ）
            print("あいこ")
        elif cp2 == 1:     # 23: ぱー vs ちょき vs ちょき  → 負けた
            print("負けた")
        elif cp2 == 2:     # 24: ぱー vs ちょき vs ぱー    → 負けた
            print("負けた")
    elif cp1 == 2:
        if cp2 == 0:       # 25: ぱー vs ぱー   vs ぐー    → 勝った
            print("勝った")
        elif cp2 == 1:     # 26: ぱー vs ぱー   vs ちょき  → 負けた
            print("負けた")
        elif cp2 == 2:     # 27: ぱー vs ぱー   vs ぱー    → あいこ
            print("あいこ")
```

## プログラムの解説

### 3段階ネスト

二人版では2段階だった `if` の入れ子が、三人版では **3段階**になりました。

```
if player == 0:         ← 第1段：プレイヤーの手
    if cp1 == 0:        ← 第2段：コンピュータ1の手
        if cp2 == 0:    ← 第3段：コンピュータ2の手
            print(...)
```

インデント（字下げ）が深くなるほど「さらに細かい条件」を表しています。

### プログラムの行数

27パターンに対応する `print` が27行あります。コンピュータの台数が増えるたびに3倍になっていきます。

| 版 | パターン数 | print の数 |
|---|---|---|
| 二人版 | 9 | 9 |
| 三人版 | 27 | 27 |
| 四人版（仮） | 81 | 81 |

このままでは増え続けてしまいます。次のチャプターで短くする方法を考えます。
