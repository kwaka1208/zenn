## どうすれば短くなる？

前のチャプターのプログラムは、9つのケースを全部 `if` 文で書き並べていました。これを短くするには、**パターンに共通する部分を見つける**のがポイントです。

表をもう一度見てみましょう。

| # | プレイヤー | コンピュータ | 結果 |
|---|---|---|---|
| 1 | ぐー（0） | ぐー（0） | **あいこ** |
| 2 | ぐー（0） | ちょき（1） | 勝った |
| 3 | ぐー（0） | ぱー（2） | 負けた |
| 4 | ちょき（1） | ぐー（0） | 負けた |
| 5 | ちょき（1） | ちょき（1） | **あいこ** |
| 6 | ちょき（1） | ぱー（2） | 勝った |
| 7 | ぱー（2） | ぐー（0） | 勝った |
| 8 | ぱー（2） | ちょき（1） | 負けた |
| 9 | ぱー（2） | ぱー（2） | **あいこ** |

**「あいこ」は、プレイヤーとコンピュータの手が同じとき**（1・5・9番）に限られます。これは `player == cp1` という1つの条件で表せます！

## ステップ1：あいこを先に判定する

「あいこ」を先に処理してしまえば、残りは「勝った」か「負けた」の2択になります。

```python
import random

hands  = {0: "ぐー", 1: "ちょき", 2: "ぱー"}
player = random.randint(0, 2)
cp1    = random.randint(0, 2)

print(f"プレイヤーの手: {hands[player]}")
print(f"コンピュータの手: {hands[cp1]}")
print("結果: ", end="")

# まずあいこを判定する
if player == cp1:
    print("あいこ")

# 残りのパターンをプレイヤーの手ごとに判定する
if player == 0:           # プレイヤーがぐー
    if cp1 == 1:
        print("勝った")   # ちょきに勝つ
    elif cp1 == 2:
        print("負けた")   # ぱーに負ける

if player == 1:           # プレイヤーがちょき
    if cp1 == 0:
        print("負けた")   # ぐーに負ける
    elif cp1 == 2:
        print("勝った")   # ぱーに勝つ

if player == 2:           # プレイヤーがぱー
    if cp1 == 0:
        print("勝った")   # ぐーに勝つ
    elif cp1 == 1:
        print("負けた")   # ちょきに負ける
```

各グループの `if` 文が2つに減りました（あいこを除いたので、各グループ3→2パターン）。

## ステップ2：さらに条件式にまとめる

「勝った」になるパターンをよく見ると、3つだけです。

| プレイヤー | コンピュータ |
|---|---|
| ぐー（0） | ちょき（1） |
| ちょき（1） | ぱー（2） |
| ぱー（2） | ぐー（0） |

この3つを `or` でつなげれば1行で書けます。「勝ち以外のあいこでないケース」はすべて「負け」なので `else` で処理できます。

```python
import random

hands  = {0: "ぐー", 1: "ちょき", 2: "ぱー"}
player = random.randint(0, 2)
cp1    = random.randint(0, 2)

print(f"プレイヤーの手: {hands[player]}")
print(f"コンピュータの手: {hands[cp1]}")
print("結果: ", end="")

if player == cp1:
    print("あいこ")
elif (player == 0 and cp1 == 1) or (player == 1 and cp1 == 2) or (player == 2 and cp1 == 0):
    print("勝った")
else:
    print("負けた")
```

`if` / `elif` / `else` の **3行**になりました！

## ステップ2のポイント

| 条件 | 意味 |
|---|---|
| `player == cp1` | 手が同じ → あいこ |
| `(player==0 and cp1==1) or ...` | 勝ちの3パターンのどれかに当てはまる |
| `else` | それ以外 → 負け |

もとの9パターンの `if` が、たった3つの分岐で表現できました。

## まとめ

- **共通パターンを見つけて先に処理する**（あいこ）
- **「それ以外はすべて〇〇」という形にする**（else で負けをまとめる）
