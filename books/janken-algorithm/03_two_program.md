## 表をそのままプログラムにしよう

前のチャプターで作った9通りの表を、そのまま `if` 文で書き起こします。

「プレイヤーの手ごとにグループ分け → その中でコンピュータの手を判定」という2段階の構造を **`if` 文の入れ子（ネスト）** で表現します。

## プログラム

```python
import random

# 手の数値を文字列に変換するための辞書（表示用）
hands  = {0: "ぐー", 1: "ちょき", 2: "ぱー"}
player = random.randint(0, 2)  # プレイヤーの手を 0〜2 でランダムに決定
cp1    = random.randint(0, 2)  # コンピュータの手を 0〜2 でランダムに決定

print(f"プレイヤーの手: {hands[player]}")
print(f"コンピュータの手: {hands[cp1]}")
print("結果: ", end="")

# プレイヤーがぐー（0）
if player == 0:
    if cp1 == 0:       # 1: ぐー vs ぐー
        print("あいこ")
    elif cp1 == 1:     # 2: ぐー vs ちょき
        print("勝った")
    elif cp1 == 2:     # 3: ぐー vs ぱー
        print("負けた")

# プレイヤーがちょき（1）
if player == 1:
    if cp1 == 0:       # 4: ちょき vs ぐー
        print("負けた")
    elif cp1 == 1:     # 5: ちょき vs ちょき
        print("あいこ")
    elif cp1 == 2:     # 6: ちょき vs ぱー
        print("勝った")

# プレイヤーがぱー（2）
if player == 2:
    if cp1 == 0:       # 7: ぱー vs ぐー
        print("勝った")
    elif cp1 == 1:     # 8: ぱー vs ちょき
        print("負けた")
    elif cp1 == 2:     # 9: ぱー vs ぱー
        print("あいこ")
```

## プログラムの解説

### ランダムに手を決める

```python
player = random.randint(0, 2)
```

`random.randint(0, 2)` は、0・1・2 の中からランダムに1つの整数を返します。これがじゃんけんの手（0=ぐー、1=ちょき、2=ぱー）に対応します。

### 辞書で数字を名前に変換する

```python
hands = {0: "ぐー", 1: "ちょき", 2: "ぱー"}
print(f"プレイヤーの手: {hands[player]}")
```

`hands[player]` のように書くと、数字から対応する手の名前を取り出せます。プログラム内部では数字を使い、表示するときだけ名前に変換します。

### 2段階の if 文

外側の `if` でプレイヤーの手を判定し、内側の `if` でコンピュータの手を判定します。

```
if player == 0:        ← 外側：プレイヤーがぐーかどうか
    if cp1 == 0:       ← 内側：コンピュータがぐーかどうか
        print("あいこ")
    elif cp1 == 1:
        print("勝った")
    elif cp1 == 2:
        print("負けた")
```

プレイヤーの3グループ × コンピュータの3パターン = **9つの `print` 文**がそれぞれのケースに対応しています。
