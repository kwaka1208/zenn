## 数字で手を表す

Pythonのプログラムで「ぐー・ちょき・ぱー」を扱うとき、この本では次のように**数字**に対応させます。

| 数字 | 手 |
|---|---|
| 0 | ぐー |
| 1 | ちょき |
| 2 | ぱー |

数字にすることで、`random.randint(0, 2)` を使ってランダムに手を決めることができます。

```python
import random
player = random.randint(0, 2)  # 0, 1, 2 のどれかをランダムに選ぶ
```

また、数字から手の名前を取り出すために**辞書（dict）**を使います。

```python
hands = {0: "ぐー", 1: "ちょき", 2: "ぱー"}
print(hands[0])  # → ぐー
print(hands[1])  # → ちょき
print(hands[2])  # → ぱー
```

辞書は `{キー: 値}` の形式で、キーを指定すると対応する値を取り出せます。ここでは数字（0・1・2）がキー、手の名前（"ぐー"など）が値になっています。
