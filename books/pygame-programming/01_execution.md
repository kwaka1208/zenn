---
title: 開発の準備と実行方法
---
Pythonのプログラムを作成し、実行するための基本的な操作を確認しておきましょう。

## フォルダの作成とVS Codeで開く
プログラムを作成する前に、ファイルを保存するための専用のフォルダ（作業フォルダ）を作成しましょう。

1. パソコンのデスクトップやドキュメントなど、好きな場所に新しいフォルダを作成します。
   - フォルダ名は「pygame_study」など、半角英数字にすることをおすすめします。
2. Visual Studio Code（VS Code）を起動します。
3. VS Codeのメニューから「ファイル」→「フォルダーを開く...」を選択し、先ほど作成したフォルダを選択します。

これで、作成したフォルダの中でプログラムを書いていく準備が整いました。

## プログラムの保存方法
プログラムを入力したら、必ず保存する必要があります。ショートカットキーを使うのが一番効率的です。

- **Windows / Chromebook**: `Ctrl` + `S`
- **Mac**: `Command` + `S`

プログラムは最後にまとめて保存するのではなく、数行書くたびにこまめに保存する癖をつけましょう。

## プログラムの実行方法
Visual Studio Code（VS Code）のターミナル機能を使って実行します。

1. VS Codeのメニューから「ターミナル」→「新しいターミナル」を選択します。
2. 画面下部に表示されたターミナルに、以下のコマンドを入力して `Enter` キーを押します。

### Windowsの場合
```powershell
python game.py
# または
py game.py
```

### Mac / Chromebookの場合
```bash
python3 game.py
```

実行すると新しいウィンドウが開き、プログラムが動き出します。

## プログラムの終了方法
実行中のプログラムを止めるには2つの方法があります。

1. **ゲーム画面を閉じる**: ウィンドウの「×」ボタンを押します。
2. **ターミナルで強制終了**: ターミナル画面をクリックして選択した状態で、`Ctrl` + `C` を押します。

プログラムが動かなくなったり（フリーズしたり）、ループから抜け出せなくなった場合は、`Ctrl` + `C` で強制終了してください。
