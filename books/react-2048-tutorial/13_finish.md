## このチュートリアルで学んだこと

### Reactの基礎

| 概念 | 内容 |
|---|---|
| **コンポーネント** | UIを部品に分けて管理する考え方 |
| **JSX** | JavaScriptの中にHTMLのような記述を書く構文 |
| **Props** | コンポーネントに外からデータを渡す仕組み |
| **useState** | 状態を管理して画面を自動更新する仕組み |
| **useEffect** | 描画後に副作用を実行する仕組み（キー入力の登録など） |
| **条件付きレンダリング** | `{条件 && <JSX>}` で状況に応じて表示を切り替える |

### JavaScriptのテクニック

| テクニック | 使った場面 |
|---|---|
| `map` で配列を変換 | ボードの描画、移動ロジック |
| `filter` で要素を絞る | 0を除いてタイルを詰める |
| スプレッド構文 `[...arr]` | 配列のコピーを作る |
| オブジェクトとキーの対応 | `moves[e.key]` で関数を選ぶ |
| 転置（transpose） | 上下移動を左右移動で実現する |

## 次のステップ

ゲームは完成しましたが、まだまだ改善できることがあります。ぜひ挑戦してみてください。

### 機能追加のアイデア

**ベストスコアを記録する**
`localStorage` を使って、ページを閉じてもベストスコアが残るようにしましょう。

```js
// 保存
localStorage.setItem('bestScore', score);

// 読み込み
const best = Number(localStorage.getItem('bestScore') ?? 0);
```

**2048達成時のメッセージ**
ボードの中に2048のタイルが出たら「クリア！」と表示してみましょう。

```js
const hasWon = board.some(row => row.some(v => v >= 2048));
```

**スマートフォン対応（タッチ操作）**
スワイプ操作に対応させると、スマートフォンでも遊べるようになります。`touchstart` と `touchend` イベントを使って、スワイプの方向を検出しましょう。

**アニメーション**
CSSの `transition` や `@keyframes` を使って、タイルが移動するアニメーションをつけてみましょう。より滑らかなゲーム体験になります。

## Reactをさらに学ぶには

このチュートリアルで学んだのはReactのほんの一部です。次は以下のトピックに進むと理解が深まります。

- **useReducer**：複雑な状態管理を整理するフック
- **React Router**：複数のページを持つアプリを作る
- **カスタムフック**：ロジックを再利用可能な形にまとめる
- **TypeScript**：型をつけてバグを減らす

Reactの公式ドキュメント（react.dev）は日本語にも対応していて、丁寧に書かれているのでおすすめです。
