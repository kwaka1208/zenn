## このチュートリアルで学んだこと

### Reactの基礎

| 概念 | 内容 |
|---|---|
| **コンポーネント** | UIを部品に分けて管理する考え方 |
| **TSX** | TypeScriptの中にHTMLのような記述を書く構文 |
| **Props** | コンポーネントに外からデータを渡す仕組み |
| **useState** | 状態を管理して画面を自動更新する仕組み |
| **useEffect** | 描画後に副作用を実行する仕組み（キー入力の登録など） |
| **条件付きレンダリング** | `{条件 && <TSX>}` で状況に応じて表示を切り替える |

### TypeScriptの基礎

| 概念 | 内容 |
|---|---|
| **型アノテーション** | 変数・引数・戻り値に型を指定する（`: number` など） |
| **interface** | オブジェクトの形を定義する |
| **type エイリアス** | 型に名前をつける（`type Board = number[][]` など） |
| **Record型** | キーと値の型を指定したオブジェクト |
| **基本型** | `number`・`string`・`boolean`・`number[]` など |

### TypeScriptを使ってよかったこと

今回のゲームを作りながら、TypeScriptの恩恵を実感できる場面がいくつかありました。

- `Board` 型のおかげで「何の配列か」が一目でわかる
- `MoveResult` 型で関数の戻り値の形が保証される
- `KeyboardEvent` 型で `e.key` が使えることが事前にわかる
- 型が合わないコードはエラーで気づける

## 次のステップ

ゲームは完成しましたが、まだまだ改善できることがあります。ぜひ挑戦してみてください。

### 機能追加のアイデア

**ベストスコアを記録する**
`localStorage` を使って、ページを閉じてもベストスコアが残るようにしましょう。

```ts
localStorage.setItem('bestScore', String(score));
const best = Number(localStorage.getItem('bestScore') ?? '0');
```

**2048達成時のメッセージ**
ボードの中に2048のタイルが出たら「クリア！」と表示してみましょう。

```ts
const hasWon: boolean = board.some(row => row.some(v => v >= 2048));
```

**スマートフォン対応（タッチ操作）**
スワイプ操作に対応させると、スマートフォンでも遊べるようになります。`TouchEvent` 型を使って `touchstart` と `touchend` イベントを処理しましょう。

**アニメーション**
CSSの `transition` や `@keyframes` を使って、タイルが移動するアニメーションをつけてみましょう。

### さらにTypeScriptを深めるには

このチュートリアルで使ったのはTypeScriptの基本的な機能です。次は以下のトピックに進むと理解が深まります。

- **ジェネリクス**：型を引数のように扱う高度な機能
- **ユニオン型**：`'left' | 'right' | 'up' | 'down'` のように複数の型をまとめる
- **型ガード**：条件分岐の中で型を絞り込む
- **strictモード**：より厳格な型チェックを有効にする

### Reactをさらに学ぶには

- **useReducer**：複雑な状態管理を整理するフック
- **React Router**：複数のページを持つアプリを作る
- **カスタムフック**：ロジックを再利用可能な形にまとめる

Reactの公式ドキュメント（react.dev）は日本語にも対応していて、TypeScriptを使った例も多く掲載されています。
