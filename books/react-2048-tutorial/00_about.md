## このチュートリアルについて

このチュートリアルでは、人気パズルゲーム「2048」をReact.jsで作りながら、Reactの基本的な使い方を学びます。

「ゲームを作る」という具体的な目標があると、プログラミングの学習は楽しく進めやすくなります。コードを少しずつ書き加えていくことで、最終的にブラウザで動くゲームが完成します。

### 完成するもの

チュートリアルを最後まで進めると、次のような2048ゲームが完成します。

- 4×4マスのゲームボードが表示される
- 矢印キーでタイルを上下左右に動かせる
- 同じ数字のタイルがぶつかると合体して2倍の数字になる
- スコアが表示される
- タイルが動けなくなるとゲームオーバーになる

### 前提知識

このチュートリアルは**Reactをはじめて学ぶ方**を対象にしています。以下の知識があると進めやすいです。

- HTMLのタグ（`<div>`、`<p>`など）を見たことがある
- CSSで色や大きさを変えたことがある
- JavaScriptで変数・関数・配列を書いたことがある

JavaScriptをほとんど書いたことがない場合は、先にJavaScriptの基礎を学んでから戻ってくることをおすすめします。

## 2048とは

「2048」は2014年にGabriele Cirulliによって公開された、シンプルながらも奥深いパズルゲームです。

### ゲームのルール

1. **4×4のボード**の上にタイルが並んでいます
2. **矢印キー**を押すと、すべてのタイルがその方向に動きます
3. **同じ数字のタイルが隣り合わせになると合体**して、2倍の数字になります（例：2と2が合体すると4になる）
4. タイルが動くたびに、空きマスに**2または4のタイル**がランダムで出現します
5. **2048のタイルを作れば目標達成**です
6. タイルがボードを埋め尽くして**動けなくなるとゲームオーバー**です

### なぜ2048をReactで作るの？

2048は「ゲームの状態（ボードの数字の並び）が変わるたびに画面を更新する」という仕組みが必要です。これはReactがもっとも得意とすることのひとつです。

- ボードの状態が変わる → Reactが変わった部分だけ画面を更新する

このシンプルな流れを体験することで、Reactの考え方を自然に身につけられます。
